## InputMode

`input` 태그에는 `inputmode` 라는 attribute가 있음

모바일에서 어떤 형식의 키보드가 보이게 할지 정해주는 attribute

- none
- text (default)
- decimal
- numeric
- tel
- search
- email
- url
